# Screen Recorder Extension

Chrome Extension для записи экрана в максимальном качестве без курсора и уведомлений.

## Возможности

- ✅ Запись всей вкладки или выбранной области
- ✅ Максимальное качество записи (до 50 Mbps, 60 fps)
- ✅ Настройка длительности записи
- ✅ Автоматическое сохранение в формате WebM
- ✅ Без курсора и системных уведомлений на записи
- ✅ Включение/выключение расширения одной кнопкой

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode)
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `screen-recorder-extension`

## Использование

1. **Включите расширение** - переключите тумблер "Extension Enabled"
2. **Настройте параметры:**
   - Длительность записи (в секундах)
   - Режим записи: вся вкладка или выбранная область
   - Имя файла
   - Качество записи
3. **Нажмите "Start Recording"**
4. Запись начнется автоматически и остановится через указанное время
5. Файл сохранится в папку загрузок

## Режимы записи

### Вся вкладка
Записывает всю активную вкладку браузера.

### Выбранная область
1. Выберите режим "Selected Area"
2. Нажмите "Choose Area"
3. На странице выберите область для записи (перетащите мышью)
4. Нажмите Escape для отмены

## Качество записи

- **Maximum** (50 Mbps, 60 fps) - максимальное качество
- **High** (30 Mbps, 60 fps) - высокое качество
- **Medium** (15 Mbps, 30 fps) - среднее качество

## Технические детали

- Использует `chrome.tabCapture` API для захвата вкладки
- MediaRecorder API для записи в WebM формат
- VP9 кодек для максимального качества
- Автоматическое сохранение через Chrome Downloads API

## Разрешения

Расширение требует следующие разрешения:
- `activeTab` - доступ к активной вкладке
- `storage` - сохранение настроек
- `tabCapture` - захват содержимого вкладки
- `downloads` - сохранение записанных файлов
